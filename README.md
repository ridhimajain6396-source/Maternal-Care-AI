# 🤱 Maternal Care AI

A smart maternal health and fitness platform designed for rural areas.

## 🌍 Problem
Many rural women lack access to proper healthcare, guidance, and government schemes during pregnancy.

## 🎯 Solution
This app helps women from pregnancy to post-delivery with:
- Week-wise pregnancy tracking
- Medicine reminders
- Risk prediction
- Government scheme awareness

## 💊 Features
- Trimester-based health guidance
- Nutrition suggestions
- Checkup reminders
- Health risk alerts

## 🛠 Tech Stack
- Python
- Flask
- SQLite
- Machine Learning

## ▶️ How to Run
1. Install Python
2. Run:
   python app.py
