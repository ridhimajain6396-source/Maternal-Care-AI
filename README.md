# 🤱 Smart Maternal Health Platform

## About
AI-powered maternal health tracking platform for rural India.
Tracks pregnancy from Week 1 to 6 months post-delivery.

## Features
- 📝 Patient Registration
- 📊 Daily Health Tracking
- 💊 Medicine Schedule & Compliance
- 🥗 Nutrition Guide & Analysis
- 🔮 ML Risk Prediction
- 🏛️ Government Scheme Finder
- 📈 Analytics Dashboard
- 🧠 Postpartum Depression Screening

## Tech Stack
- Python
- Streamlit
- Scikit-learn (Machine Learning)
- SQLite (Database)
- Plotly (Charts)

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
